# GDChat server
Server for [GDChat](https://github.com/bit0r1n/GDChat)

## Setup (Linux, Windows Command Prompt)

```bash
npm i

echo "PORT=replace_with_your_free_port" > .env

node .
```
## Setup (PowerShell)

```powershell
npm i

New-Item .env | Set-Content .env "PORT=replace_with_your_free_port"

node .
```
